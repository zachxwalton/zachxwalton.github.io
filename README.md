# Student Resume — Single Page (Tailwind)

This repo is a **GitHub Pages–ready** single-page student resume site styled with **Tailwind CSS** (via CDN).

## Edit me

Open `index.html` and replace:

- Name, location, focus
- Education details
- Project titles/descriptions + links
- Skills
- Email + social links

## Run locally

Any static server works. For example:

```bash
python -m http.server 5173
```

Then open `http://localhost:5173`.

## Deploy to GitHub Pages

- Push to GitHub
- In **Settings → Pages**
  - **Source**: Deploy from a branch
  - **Branch**: `main` / `(root)`

GitHub Pages will serve `index.html` automatically.
